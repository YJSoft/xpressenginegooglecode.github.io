계정 소개

XpressEngineGoogleCode 계정은, Google Code의 모든 XpressEngine 관련 Project를 import하여 저장하고 있습니다. 권한이 있는 경우 저장소 이전 혹은 삭제를 요청하실 수 있습니다. 단, 저장소 삭제를 요청하시려면 동일 저장소의 Github 이전본이 존재해야 합니다. 이경우, 해당 저장소는 삭제된후 재생성되며, 새로운 저장소의 링크를 안내하게 됩니다.

요청은?

모든 이전/삭제 요청은 xecode@yjsoft.ml 로 받습니다. 이외 다른 이메일 주소로 보내실 경우 처리되지 않음을 명심해 주시기 바랍니다.

권한 증명 방법

프로젝트에 관리자로 등록된 gmail 주소로 보내 주시면 즉시 처리됩니다. 하지만 보내시는 메일 주소가 프로젝트 관리자가 아닌 경우, 프로젝트 관리자로 등록된 이메일로 확인 메일이 발송될 수 있습니다.

Account Introduction

All repos of this github accounts are from Google code. I DO NOT OWN NONE OF THIS REPOS. if you want to make requests(such as repo transport or delete), please send mail to xecode@yjsoft.ml . If you want to delete repo, you should fork or import google code repo by yourself, and add a repo link to request email. requested repo will be deleted and new repo(by you) address will be noticed. Please be advised that you should send email from your google project admin gmail, otherwise an auth email would be sent to admin gmail address.
